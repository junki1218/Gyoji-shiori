# 宿泊行事の栞（しおり）作成アプリケーション 完成レポート

修学旅行や宿泊行事の栞を作成できるReactアプリケーションを構築しました！

> [!TIP]
> アプリケーションを起動するには、`Gyoji-shiori` ディレクトリで以下のコマンドを実行し、表示されたURLにアクセスしてください。
> ```bash
> cd ~/デスクトップ/antigravity-workspace/Gyoji-shiori
> npm run dev
> ```

## 実装内容のハイライト

### 1. モダンで美しいデザイン
- **Glassmorphism (ガラス効果)**: アプリケーションのメインエリアには、美しい背景グラデーションを透かして見せるすりガラス風のデザインを採用しました。
- **洗練されたタイポグラフィ**: `Inter` フォントと `M PLUS Rounded 1c` を組み合わせ、モダンでありながら親しみやすい文字スタイルに仕上げています。
- **マイクロアニメーション**: ホバー時のボタンの浮き上がりや、ページ遷移時のフェードインアニメーションを実装し、触り心地の良いUIになっています。

### 2. コンポーネント・機能構成
React Routerを利用し、全15項目＋プレビューの画面をシームレスに行き来できるようにしました。

| ページ名 | 実装した機能 |
| --- | --- |
| **日時・行き先・目標** | テキスト入力やカレンダー(Date)ピッカーを使った基本的な情報入力 |
| **もちもの・スケジュール** | リストの動的追加・削除機能、重要(必須)フラグのトグル |
| **座席（バス・電車）** | 列数と席数を入力することで、リアルタイムに座席表のレイアウト（マス目）が生成され、そこに名前を入力できる機能 |
| **泊まる部屋** | 定員を設定し、定員に達するまでメンバーを追加できるUI |
| **お小遣い帳** | 予算を設定し、使った金額を入力するとリアルタイムに残金が計算される機能 |
| **かかり** | 役割と担当者をペアで動的に追加できる機能 |
| **フリーページ (11〜15)** | 見出し（タイトル）と本文を自由に変更・入力できる汎用ページ |

### 3. データ永続化と印刷（プレビュー）機能
- **LocalStorage連携**: 入力したすべてのデータはブラウザに自動的に保存されます。ページをリロードしたり、翌日開いたりしてもデータが消えることはありません。
- **リセット機能**: サイドバー下部にある「データリセット」ボタンで、初期状態に戻すことも可能です。
- **プレビュー画面**: 「完成プレビュー」ページでは、入力した全データが1つの「しおり」フォーマットにまとめられて表示されます。この画面で「印刷する」ボタンを押すことで、サイドバーなどが消え、きれいなPDFとして保存・印刷が可能です。
